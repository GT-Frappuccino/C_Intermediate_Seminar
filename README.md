## 실습 시 유의사항
* 코드에 젹혀있는 변수명, 함수명 등은 되도록이면 수정하지 않을 것을 권함.  

* 코드 작성 부분
```
code_here() {
    /*** Your Code Here ***/
}      
```  

* 제거  
delete 주석이 달려있는 부분. 테스트 용으로 써둔 코드. 지우면 됨.
```
delete_this_line() // delete
```
